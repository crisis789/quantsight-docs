# Как настроить алерты

Все индикаторы QuantSight поддерживают встроенные алерты.

1. Нажмите на значок **⏰** на панели алертов
2. Выберите условие (например, «BUY Signal»)
3. Выберите способ уведомления: **Telegram, Email, SMS, Discord, Webhook**
4. Сохраните

{% hint style="success" %}
**Круглосуточные алерты** — получайте сигналы 24/7, даже когда не за компьютером!
{% endhint %}

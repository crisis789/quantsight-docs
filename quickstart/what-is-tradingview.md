# Что такое TradingView?

**TradingView** — крупнейшая в мире бесплатная платформа для графиков и анализа рынков.

Трейдеры и инвесторы используют TradingView для просмотра криптовалют, акций, форекс, фьючерсов и других рынков.

{% hint style="info" %}
TradingView: [cn.tradingview.com](https://cn.tradingview.com)
{% endhint %}

## Нужна ли платная подписка TradingView?

**Нет.** Бесплатный аккаунт TradingView позволяет загружать до 2 индикаторов одновременно.

Для использования всех функций QuantSight переключайте индикаторы по мере необходимости. Платная подписка TradingView не требуется.

{% hint style="success" %}
Бесплатный TradingView + QuantSight = полноценная аналитическая система!
{% endhint %}

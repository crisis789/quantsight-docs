# Как установить индикаторы

После покупки подписки QuantSight:

1. **Откройте график** на TradingView
2. Нажмите **«Индикаторы»** на верхней панели
3. Перейдите во вкладку **«Только по приглашению»**
4. Выберите нужный индикатор QuantSight
5. Нажмите **★** для быстрого доступа

{% hint style="info" %}
После оплаты отправьте ваш TradingView username на [info@quantsight.io](mailto:info@quantsight.io). Доступ активируется в течение 24 часов.
{% endhint %}

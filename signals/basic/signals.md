# Базовый — Сигналы и панель

## Панель индикаторов

- **Волатильность 🚦** — на основе среднего направленного движения
- **RSI 🚦** — уровни перекупленности/перепроданности
- **Мульти-таймфрейм 🚀** — тренды на разных таймфреймах без переключения

## Чувствительность

- **Низкая** → быстрые сигналы для краткосрочной торговли (больше шума)
- **Высокая** → стабильные сигналы для долгосрочных трендов

## Типы сигналов

- ★ — трендовый сигнал
- Без ★ — обычный сигнал

{% endhint %}

# Сигнальные системы

QuantSight предлагает три уровня сигнальных систем:

| Система | Уровень | Особенности |
|---------|---------|-------------|
| **Базовый** | Начинающий | Простые и быстрые сигналы |
| **PRO** | Профессионал | Многомерное квантование, AI-адаптация |
| **MAX** | Эксперт | Передовое ML-моделирование, 0-5 датасетов |

{% hint style="info" %}
Выберите **одну** систему как основную. Все три подтверждают друг друга!
{% endhint %}
